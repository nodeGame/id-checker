Part 6: Instructions


In this part, you will work on an item recognition task.

  • On the next page, you will see a set of 30 different items which you should memorize.
  • Each item is only displayed for one second.
  • Afterwards, you will see another set of pictures.
  • For each item, we will ask you whether it was part of the first set.
  • For each correct response, you earn a bonus of $-.
At the end of the task, we will let you know how many items you have correctly identified.

To start this task, please click the Next button at the top of this page.
